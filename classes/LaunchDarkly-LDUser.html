LDUser.php : 24

Contains specific attributes of a user browsing your site.

LDUser supports only a subset of the behaviors that are available with the newer LDContext type. An LDUser is equivalent to an individual LDContext that has a kind of LDContext::DEFAULT_KIND; it also has more constraints on attribute values than LDContext does (for instance, built-in attributes such as LDUserBuilder::email() can only have string values). Older LaunchDarkly SDKs only had the LDUser model, and the LDUser type has been retained for backward compatibility, but it may be removed in a future SDK version. Therefore, developers are recommended to migrate toward using LDContext.

The only mandatory property property is the key, which must uniquely identify each user. For authenticated users, this may be a username or e-mail address. For anonymous users, it could be an IP address or session ID.

Use LDUserBuilder to construct instances of this class.
